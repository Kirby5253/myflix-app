This is an API that will distribute the information about movies I've seen.

Request URL HTTP Method Request body data format Response body data format
Welcome Screen for myFlix / GET None Text with a Welcome message
Get all movies by title /movies GET None A JSON holding a list of each movie title currently in the database.
Get movie details /movies/[Title] GET None A JSON holding details about a movie containing the title, movie description, related genre(name and description), director(name and bio), image, actors, and unique id. Example:
{
_id: "5ee3ba9febd72a15cf02a299",

Title: "Iron Man",

Description:"A billionaire weapons maker is kidnapped and has to force his way out of imprisonment by designing a super suit that leads him to becoming a superhero in his own way.",

Genre:{
Name: "Science Fiction"
Description: "Science fiction uses fictional-based displays of phenomena not fully accepted by main-steam scientists."
},

Director: {
Name: "Jon Favreau",
Bio: "American actor, director, producer, and screenwriter."
Birth: "1966"
},

ImagePath: "ironman.png",

Actors: ["Robert Downey Jr.", "Jon Favreau", "Gwyneth Paltrow", "Terrence Howard"],

Featured: true
}
Get all genres by title /genres GET None A JSON holding a list of each unique movie genre
Get genre details /genres/[Name] GET None A JSON holding details about a movie genre containing the name, and genre description. Example:
{
Name: "Action",
Description: "Action films generally involve violence, fighting, and physical activities."
}
Get a list of directors by name /directors GET None A JSON holding a list of each movie director
Get director details /directors/[Name] GET None A JSON holding details about a movie director containing name, movies directed, and birth date. Example:
{
"Name": "Jon Favreau",
"Bio": "American actor, director, producer, and screenwriter.",
"Birth": "1965"
}
Create a new user account /users POST A JSON with the username, password, email, and dob structured like:
{
"Username": "example-username",
"Password": "1234"
"Email": "kirby_jake@gmail.com"
"Birth_Date": "1991-07-12"
} 		A JSON object holding data about the user that was added:
{
"Username": "example-username",
"Password": "1234",
"Email": "kirby_jake@gmail.com",
"Birth_Date": "1991-07-12",
"Favorite_Movies": [] }
Get a list of all users /users GET None A JSON holding a list of each user with their corresponding information.
Get account details /users/[Username] GET None A JSON holding details about a user account containing name, username, password, email, and dob structured like:
{
"Username": "example-username",
"Password": "1234",
"Email": "kirby_jake@gmail.com",
"Birth_Date": "1991-07-12",
"Favorite_Movies": [] }
Delete user account /users/[Username] DELETE None A text message stating that the user account was deleted.
Edit user account /users/[Username] PUT A JSON with all of the updated and current information structured like:
{
"Username": "updated-username",
"Password": "newpassword",
"Email": "kirby_jake@newgmail.com",
"Birth_Date": "1991-07-12",
} 		A JSON of the updated user account:
{
"Username": "updated-username",
"Password": "newpassword",
"Email": "kirby_jake@newgmail.com",
"Birth_Date": "1991-07-12",
"Favorite_Movies": []
}
Add movie to favorites /users/[Username]/Movies/[MovieID] POST None A JSON of the updated user account with the newly added favorite movie.
{
"Username": "updated-username",
"Password": "newpassword",
"Email": "kirby_jake@newgmail.com",
"Birth_Date": "1991-07-12",
"Favorite_Movies": [ "5ee3bec2ebd72a15cf02a2a1"]
}
Remove movie from favorites /users/[Username]/Movies/[MovieID] PUT None A JSON of the updated user account without the removed favorite movie.
{
"Username": "updated-username",
"Password": "newpassword",
"Email": "kirby_jake@newgmail.com",
"Birth_Date": "1991-07-12",
"Favorite_Movies": []
}
